Rekomendasi

Implikasi Kebijakan Berbasis Machine Learning

Temuan analisis SHAP dan clustering menghasilkan rekomendasi kebijakan berbasis bukti untuk penguatan ketahanan pangan Jawa Tengah.

🌾
1. Intervensi NCPR Prioritas
  • Fokus subsidi pangan di 509 kecamatan L-FSG
  • Program diversifikasi konsumsi pangan lokal
  • Penguatan BULOG di wilayah NCPR tinggi
  • Monitoring real-time ketersediaan kalori
🏥
2. Penguatan Layanan Kesehatan
  • Tambah tenaga kesehatan di L-FSG daerah terpencil
  • Program stunting terintegrasi dengan pangan
  • Mobile health unit untuk kecamatan terisolasi
  • Penguatan Posyandu berbasis data IKP
💧
3. Infrastruktur Dasar
  • Program air bersih di wilayah Tanpa Air Bersih > 50%
  • Sanitasi berbasis komunitas (PAMSIMAS)
  • Investasi listrik & infrastruktur pedesaan
  • Konektivitas akses pasar pangan
🎓
4. Pemberdayaan Perempuan
  • Program kesetaraan pendidikan perempuan
  • Lama sekolah perempuan berkorelasi positif IKP
  • Pelatihan gizi & ketahanan pangan rumah tangga
  • Pemberdayaan ekonomi perempuan desa
🗺️ Peta Prioritas Intervensi Berdasarkan ML

Bobot prioritas intervensi dihitung dari permutation importance × proporsi klaster L-FSG per dimensi kebijakan.
